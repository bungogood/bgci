# UBGI Docs

This folder contains protocol notes plus dueller usage docs for **bgci**.

- `ubgi-v0.1-spec.md`: protocol draft.
- `format-decisions.md`: format choices and rationale.
- `dueller.md`: how to run engine-vs-engine duels and use config files.
- `community-alignment.md`: alignment/divergence notes versus community discussions.
- `repo-readiness-plan.md`: cleanup and portability plan.

Public duel config files live in `config/`:

- `config/gnubg-cli-vs-random.toml`
- `config/gnubg-cli-vs-pubeval.toml`
- `config/pubeval-vs-random.toml`
